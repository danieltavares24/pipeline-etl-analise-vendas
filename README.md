## Visão Geral e Objetivos
Este projeto foi desenvolvido com a missão de transformar dados brutos de vendas, produtos e custos em ativos estratégicos para a tomada de decisão da **LH Nautical**. A solução documenta uma jornada técnica completa, desde o saneamento de bases heterogêneas até a construção de modelos preditivos e sistemas de recomendação por afinidade.

O foco central foi garantir a **integridade dos indicadores financeiros**, corrigindo distorções cambiais e sazonais para gerar insights acionáveis e escaláveis para a diretoria.

## Stack Tecnológica
* **Linguagem:** Python
* **Manipulação e ETL:** Pandas, JSON, CSV
* **Processamento:** NumPy
* **Engenharia de Dados:** Scripts de normalização e automação de ETL
* **Machine Learning:** Scikit-Learn

## Estrutura da Solução
O projeto foi dividido em módulos para garantir a manutenção e escalabilidade do código:

1.  **Saneamento e ETL:** Integração de bases de vendas (CSV), CRM (JSON) e custos de importação (JSON).
2.  **Normalização de Dados:** Implementação de regras de negócio para correção de moedas e padronização de produtos.
3.  **Análise de Indicadores:** Criação de KPIs financeiros para monitoramento de margens e custos de importação.
4.  **Modelagem Preditiva:** Desenvolvimento de modelos para previsão de demanda e sistemas de recomendação por perfil de consumo.

## Organização do Repositório
* `Desafio_Lighthouse_Dados_&_AI.ipynb`: Documentação completa da jornada analítica e modelagem.
* `etl_custos_importacao.py`: Script dedicado ao processamento de custos e taxas.
* `solucao_questao_2_normalizacao.py`: Lógica de normalização de bases heterogêneas.
* `desafio_lighthouse_dados_&_ai.py`: Versão em script da solução final para integração.

## Conexão com a IA Industrial
A capacidade de integrar fontes de dados distintas (como CRM e Planilhas de Custos) é uma habilidade crítica na indústria. Este projeto demonstra competência em **Sistemas de Apoio à Decisão (DSS)**, onde a IA é aplicada para otimizar o fluxo financeiro e logístico — uma base fundamental para a implementação de fábricas inteligentes e otimização de supply chain.
